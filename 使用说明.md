# 🎂 3D生日祝福网站 - 使用说明

欢迎使用这个温馨的3D互动生日祝福网站！

## 📁 文件清单

确保以下文件都在正确的位置：

```
HTML/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 交互脚本（含3D蛋糕）
└── zy/
    ├── cake.png        # 背景图片
    └── bir.mp3         # 生日歌音乐
```

## 🚀 快速开始

### 方法1：直接打开
1. 找到 `index.html` 文件
2. 双击打开（会用默认浏览器打开）
3. 享受3D生日祝福体验！

### 方法2：右键选择浏览器
1. 右键点击 `index.html`
2. 选择"打开方式" → 选择您喜欢的浏览器
3. 推荐使用：Chrome、Edge、Firefox

## 🎮 交互操作指南

### 🎂 3D蛋糕操作

| 操作 | 功能 |
|------|------|
| **拖动鼠标** | 360度旋转蛋糕，查看各个角度 |
| **鼠标滚轮** | 放大/缩小蛋糕 |
| **不操作** | 蛋糕会自动缓慢旋转 |
| **触摸拖动** | 手机/平板上拖动旋转 |

### 🎵 音乐控制

- 页面加载时会**自动播放**背景音乐
- 如果浏览器阻止自动播放，点击右上角的 🎵 按钮
- 再次点击可以暂停/继续播放

### 💝 留言祝福

1. 在文本框中输入您的生日祝福
2. 点击"送出祝福 💝"按钮（或按回车键）
3. 留言会显示在下方，并伴随粒子特效

## 🔧 常见问题

### Q: 没有声音怎么办？
**A**: 
1. 检查 `zy/bir.mp3` 文件是否存在
2. 点击右上角音乐按钮
3. 检查浏览器是否静音
4. 某些浏览器需要用户交互后才能播放音频

### Q: 看不到3D蛋糕？
**A**: 
1. 确认浏览器支持 WebGL
2. 更新浏览器到最新版本
3. 检查浏览器控制台是否有错误（按F12）
4. 尝试刷新页面（Ctrl+F5）

### Q: 背景图片不显示？
**A**: 
1. 确认 `zy/cake.png` 文件存在
2. 检查文件路径是否正确
3. 尝试清除浏览器缓存

### Q: 3D蛋糕很卡顿？
**A**: 
1. 关闭其他浏览器标签页
2. 检查电脑性能
3. 尝试降低浏览器缩放比例

### Q: 手机上无法旋转？
**A**: 
1. 确保使用单指拖动
2. 避免在页面边缘操作
3. 尝试在3D蛋糕区域内拖动

## 💡 使用技巧

1. **最佳观赏角度**：稍微从上往下看，可以看到蛋糕的层次
2. **火焰效果**：仔细观察火焰的闪烁和发光
3. **自动旋转**：放开鼠标，让蛋糕自己转动
4. **缩放观察**：滚轮放大可以看到更多细节
5. **全屏体验**：按F11进入全屏模式（再按退出）
